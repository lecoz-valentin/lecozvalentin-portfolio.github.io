Mission Découverte Symfony

Contexte

Durant ma seconde année de BTS SIO option SLAM, j'ai réalisé diverses mission dans le but d'acquiérir plusieurs notions et découvrir le développement sous plusieurs angles. Dans cette mission sur le framework Symfony, il était question de découvrir le fonctionnement d'un framework et de réaliser une petite application de gestion de devis.

Définitions :
- Framework : Un framework est un ensemble d'outils et de composants logiciels organisés conformément à un plan d' architecture et des patterns, l'ensemble formant ou promouvant un « squelette » de programme, un canevas. Il est souvent fourni sous la forme d'une bibliothèque logicielle et accompagné du plan de l'architecture cible du framework. (Source : Wikipédia)

- Symfony : Symfony est un ensemble de composants PHP ainsi qu'un framework MVC libre écrit en PHP. Il fournit des fonctionnalités modulables et adaptables qui permettent de faciliter et d’accélérer le développement d'un site web. (Source : Wikipédia)

La société JARDITRAV a pour activité l'entretien de jardins pour les particuliers et les organisations. Etant contactée fréquemment pour des devis de taille de haies, elle souhaite mettre en place une application Web permettant à toute personne d'obtenir un prix pour un tel travail.

Méthode de travail

Pour commencer la mission, il a fallu dans un prermier temps installer Symfony sur la machine. Pour ça il faut regarder la vaiable d'environnement PATH et vérifier qu'il y ait : php.exe (C:\EasyPHP\binaries\php\php_runningversion)
Par la suite il faudra installer Composer sur https://getcomposer.org/Composer-Setup.exe puis l'exécuter.

On pourras alors créer le projet et commencer la mission.

Outils utilisés et commandes à retenir

- Création d'un controller : php bin/console make:controller
- Démarrer le serveur web : php bin/console server:run (Ce serveur web écoute sur le port 8000. Par conséquent, en tapant dans un navigateur l'adresse 127.0.0.1:8000, une page d'accueil de Symfony va s'afficher.)

Conclusion de la mission

Pour conclure sur cette mission, j'ai découvert l'existence des framework et en particulier Symfony.
