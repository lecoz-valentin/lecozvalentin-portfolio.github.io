Application Web de gestion de parc de véhicules de locations

Présentation de l'entreprise

Netto est une enseigne française de hard-discount alimentaire du groupement Les Mousquetaires. C'est une marque déposée par ITM Entreprises.

Contexte

Netto Aubusson propose un service de location de véhicules à ses clients pour des voyages, déménagements, ou autres. Cependant ils faisaient la gestion de ce service à la main, devenant alors une tâche non productive. C'est ainsi qu'ils nous ont fais la demande d'une application web qui permettrait de les aider dans cette tâche. Nous devions alors proposer :

- Un système de connexion sécurisé poour différencier les différents poste de l'entrprise. (Gérant, technicien, etc...)
- Une page pour enregistrer les différents véhicules qu'ils possèdent dans une base de données.
- Renseignement techniques des véhicules. (Dimensions, nombre de kilomètres.)
- Ajout, suppression et édition des opérations possibles.
- Ajout et suppression des révisions qui ont été faites.

Méthode de travail

Lors de ce projet nous étions 6, nous avons donc répartis les tâches à l'aide de Trello ainsi que de la méthode agile.

Outils utilisés

Nous avons utilisés différents outils pour mener ce projet :
• HTML, CSS, PHP pour le développement.
• Win'Design pour la modélisation des donénes.
• Trello pour la gestion des tâches et du suivis de projet.
• La méthode Agile pour nous organiser.

Conclusion du projet

Pour conclure, ce projet qui nous à été confié dès notre première année nous as permit de découvrir la gestion de projet en profondeur avec la méthode agile notamment, et de s'organiser pour le travail en groupe.
